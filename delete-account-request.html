Account & Data Deletion

Last updated: January 2026

1. Overview

This page explains how users can request deletion of their account and associated data for any mobile application published by Brainservice Incorporated.

2. How to Request Account Deletion

  1. Send an email to support [at] brainservice.ca
  2. Use the subject line: Account Deletion Request
  3. Include the following information:
    • The name of the app
    • The email address used to sign in (Google or Apple)

Once your request is received, we will verify ownership of the account and process the deletion.

3. Data That Is Deleted

Upon successful verification, the following data associated with the specified app will be permanently deleted:

  • User account and authentication identifiers
  • Email address
  • Display name
  • Profile image or avatar
  • User-generated or app-related data (such as shopping or usage data)

4. Data Retention

All personal data associated with the account is permanently deleted. No personal data is retained after deletion, except where required by law or for legitimate security or fraud-prevention purposes.

5. Processing Time

Account deletion requests are typically processed within a reasonable timeframe after verification.

6. Contact

If you have questions regarding account or data deletion, please contact:
support [at] brainservice.ca
